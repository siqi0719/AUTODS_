# Data Cleaning Project

A data cleaning system using LangChain to automatically process and clean raw data.

## Features

- Remove duplicate rows
- Handle missing values
- Handle anomalies
- Data type conversion
- Data consistency checking

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```

## Project Structure

- `main.py` - Main entry point
- `data_cleaning_agent.py` - DataCleaner Agent implementation
- `cleaned_outputs/` - Output directory for cleaned data
- `requirements.txt` - Project dependencies
- `pyproject.toml` - Python project configuration
